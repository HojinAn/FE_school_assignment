# Weniv About 페이지 마크업 클론  
---
[제주 코딩 베이스 캠프(Paullab) About](http://paullab.co.kr/about.html)

- \<header\>, \<section\>, \<footer\>로 페이지 영역 구분  
- \<header\> 내 \<h1\>은 페이지 당 1개  
- \<strong\> - 텍스트 강조  
- \<p\> - 문단 태그  
- \<section\> - 영역을 구분  
- \<article\> - 독립된 주제를 가진 영역  

**list 태그 사용 방법**
```html
<ul>
    <li></li>
    <li></li>
    <li></li>
</ul>
```
```html
<dl>
    <dt></dt>
    <dd></dd>
</dl>
```
